Projeto A3 — Conversão, Hash e Simulação FCFS

Este projeto é um sistema web desenvolvido para a disciplina A3, contendo três funcionalidades principais:

Conversão de bases numéricas

Geração de Hash SHA-256

Simulação de escalonamento FCFS (First Come, First Served)

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript puro.

Estrutura do Projeto
/ProjetoA3
│── A3.html           → Tela inicial com os nomes dos alunos
│── A3pt2.html        → Menu principal com as três funções
│── A3C.html          → Conversão de bases numéricas
│── A3H.html          → Geração de Hash SHA-256
│── A3S.html          → Simulação FCFS
│── README.md         → Documentação do projeto

Funcionalidades
1. Conversão de Bases Numéricas (A3C.html)

Permite converter valores entre:

Binário

Octal

Decimal

Hexadecimal

Recursos:

Validação automática conforme a base selecionada

Conversão instantânea

Interface visual simples e organizada

2. Gerador de Hash SHA-256 (A3H.html)

Transforma qualquer texto inserido em um hash SHA-256 utilizando a API crypto.subtle.digest() do navegador.

Recursos:

Geração de hash em tempo real

Exibição do texto convertido em formato hexadecimal

3. Simulação do Algoritmo FCFS (A3S.html)

Simula o funcionamento do algoritmo de escalonamento First Come, First Served.

Recursos:

Adição de processos (nome + tempo de burst)

Exibição da fila de processos

Cálculo de:

Ordem de execução

Tempo de espera

Tempo de retorno

Geração de tabela final

Exibição do diagrama de Gantt

Interface e Estilo

Gradientes e sombras leves

Botões arredondados

Layout responsivo

Cores definidas para cada função

Não utiliza bibliotecas externas

Como Executar

Baixe ou extraia o projeto.

Abra o arquivo A3.html em qualquer navegador moderno.

Utilize o menu para acessar as funções:

Conversão de bases

Geração de hash

Simulação FCFS

Nenhuma instalação adicional é necessária.
